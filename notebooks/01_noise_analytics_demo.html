Noise Analytics Pipeline

Anlisis automtico de ruido ambiental mediante series temporales

Este notebook presenta un ejemplo de pipeline de anlisis de datos temporales aplicado al ruido ambiental.

El objetivo no es tanto el dominio acstico, sino mostrar:

  • cmo estructurar un flujo de datos reproducible,
  • cmo extraer informacin relevante de una serie temporal,
  • y cmo detectar automticamente eventos anmalos.

Los datos utilizados son sintticos, diseados para simular escenarios reales.

Contexto del proyecto

En muchos estudios de ruido ambiental, el anlisis de datos sigue siendo manual o semi-manual (Excel, inspeccin visual, umbrales fijos).

Este proyecto propone un enfoque automatizado basado en:

  • estadsticas mviles,
  • agregaciones temporales,
  • y deteccin adaptativa de eventos.

El pipeline completo est implementado en Python y separado en mdulos.
